# UI 设计规范文档

## 功能概述

新增项目 UI 设计风格文档 `design.md`，并在 `AGENTS.md` 中添加界面设计约束，统一项目 UI 设计规范。

## 实现步骤

1. 全面探索项目现有 UI 资源：QSS 样式表、组件代码、配置项、图标资源等
2. 整理为结构化的 `design.md` 设计规范文档
3. 在 `AGENTS.md` 中新增「界面设计约束」章节，约束后续 UI 开发行为

## 设计规范覆盖范围

- 色彩体系（主题色、背景色、状态色、文字层次）
- 字体规范（字族、字号层次）
- 圆角规范（卡片 6px、按钮 5-6px）
- 间距规范（页面内边距、组件内边距）
- 组件规范（按钮、卡片、加载状态）
- 布局规范（导航栏、页面结构）
- 图标规范（双版本 SVG、Fluent 风格）
- 样式表管理（双主题 QSS、加载机制）
- 主题切换（ColorChangeable、Mica、DPI）
