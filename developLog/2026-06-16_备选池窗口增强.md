# 大乱斗备选池窗口增强

## 功能概述

增强大乱斗备选池窗口功能：
1. 点击英雄头像直接进行替换
2. 英雄头像以正方形显示（带圆角）
3. 增加进度条显示可用倒计时
4. 显示设置的期望英雄头像
5. 窗口标题改为"大乱斗英雄备选池"

## 实现步骤

### 1. 创建可点击的正方形英雄图标组件

新增 `ClickableChampionIcon` 类：
- 继承 `QFrame`
- 正方形显示，带 6px 圆角
- 支持悬停效果（边框高亮、背景变化）
- 点击时发出 `clicked(championId)` 信号
- 通过 `asyncio.ensure_future()` 异步调用 `connector.benchSwap()`

### 2. 添加进度条显示倒计时

- 使用 `QProgressBar` 显示倒计时进度
- 从 `data['timer']['adjustedTimeLeftInPhase']` 获取剩余时间
- 使用 `QTimer` 每秒更新进度条和标签

### 3. 显示期望英雄头像

- 从 `cfg.aramAutoSwapChampions` 获取期望英雄列表
- 同步加载本地缓存的英雄图标
- 在开关下方显示 28px 圆形头像

### 4. 布局优化

- 备选池英雄左对齐（QGridLayout 设置 AlignLeft | AlignTop）
- 期望英雄移到开关下方
- 调整窗口高度为 480px

### 5. 主窗口关闭时关闭小窗口

在 `closeEvent` 中添加 `self.aramBenchWindow.close()`

## 文件变更

| 文件 | 变更说明 |
|------|----------|
| `app/view/aram_bench_window.py` | 添加可点击图标、进度条、期望英雄显示 |
| `app/view/main_window.py` | 传递 timer 数据、关闭时关闭小窗口 |
| `CLAUDE.md` | 添加禁止自动提交的约束 |
